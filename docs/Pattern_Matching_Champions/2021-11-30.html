<rbuckton>

I'm not sure what you mean by "implicit ... semantics", but one option is this:

Foo[Symbol.unapply] = function (value) {
  if (value instanceof Foo) return [value];
  return null;
};

match (x) {
  when (Foo(y)) { /*...*/ } // if you want 'y'
  when (Foo()) { /*...*/ } // if you don't want 'y'
}

But that depends on whether pattern matching checks whether the when clause is exhaustive (i.e., would when ([]) match for [1, 2, 3]). If it is exhaustive, you could do something like:

match (x) {
  when (Foo(..._)) { /*don't care about the results*/ }
}
