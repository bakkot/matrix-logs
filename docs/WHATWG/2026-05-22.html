<Ana Sollano Kim>
On this, I think the following might be interesting: I few months ago I added use counters to see if people did actually change the type of <button> and <input>: https://chromium-review.googlesource.com/c/chromium/src/+/7624034. Changing type while a <button> is connected to the DOM -> ~6% vs disconnected -> ~25% of page loads. Changing type while an <input> is connected to the DOM -> ~1% vs disconnected ~70%. This + having a few people independently bring up not following the <input type> pattern and this article https://meowni.ca/posts/a-story-about-input/, informed our decision to not have dynamic changes until we hear a valid/real use case for it (which I've asked for but have not heard any yet).
